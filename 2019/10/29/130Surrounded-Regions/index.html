    130Surrounded_Regions

    Leetcode
    创建时间:2019-10-29 15:40
    阅读:

    Example:

    X X X X
    X O O X
    X X O X
    X O X X

    X X X X
    X X X X
    X X X X
    X O X X

    题目大意:寻找包含在X中的O,并把它变成X,但是在边界上的O全部保留

    找到所有在边界上的O,把它标定为G,然后如果它是G则把它变成O,如果他是X,则把它变成X,如果是O,则把它变成X

    输入board是一个二维数组
    [[“X”,”X”,”X”,”X”],[“X”,”O”,”O”,”X”],[“X”,”X”,”O”,”X”],[“X”,”O”,”X”,”X”]]
    首先可以计算出他的行和列
    行y变化 m=len(board)
    列x变化 n=len(board[0])

    所以表示y行x列的元素为board[y][x]

    定义一个寻找边界O的函数
